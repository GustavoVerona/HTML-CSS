Principais formataçoes

negrito

para colocar uma palavra em negrito usamos a tag <strong>

italico

para colocar uma palavra em italico usamos a tag <em>

Marcação de texto

Para adicionar uma marcação sobre o texto usamos a tag MARK.

Para criar um texto um pouco maior usamos a tag BIG

Tamanhos de textos

Para criar um texto um pouco menor usamos a tag SMALL

Textos deletados e inserios

para criar um texto deletado usamos a tag DEL

Para criar um texto sublinhado usamos a tag INS

Tags sup e sub

para criar um texto elevado usamos a tag SUPex:450x8

para criar um texto um pouco a baixo usamos a tag SUB ex: h2o
